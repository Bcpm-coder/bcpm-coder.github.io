# 基础数学知识（二）

### 一、试除法求约数

求出x的所有约数


#### 样例

```text
6：1，2，3，6
```


----------

### 算法 试除法
数x由一对因子相乘得到，若d可以整除x，则x/d也可以整除x，较小的因子枚举，较大的因子计算得到，因此我们只需要试除1~sqrt(x)之间的数字即可，注意判重
### $O(sqrt(n))$
#### C++ 代码
```cpp
#include <iostream>
#include <vector>
#include <cstdio>
#include <algorithm>

using namespace std;

int n;

int main()
{
    cin >> n;
    while(n --)
    {
        int x;
        cin >> x;
        
        vector<int> res;
        for(int i = 1; i <= x / i; i ++)
        if(x % i == 0)
        {
            res.push_back(i);
            if(i != x / i)
            res.push_back(x / i);
        }
        sort(res.begin(), res.end());
        
        for(auto t : res)
        cout << t << " ";
        cout << endl;
    }
}
```

### 二、求一个超级大的数x的约束个数



#### 样例 2 * 6 * 8的约数个数

```text
输入
3
2
6
8
输出
12
```
----------

### 算法
将x分解为p1^a1 · p2^a2 · p3^a3...形式
pi表示质因子
x的任意一个约束都可分解为p1^x1 · p2^x2 · p3^x3... xi取值[0, ai]
因此每一项都有ai+1种取法，乘法公式得到约数个数：
(a1 + 1) * (a2 + 1) *...* (ak + 1) 

### $O(nlog(a))$
#### C++ 代码
```cpp
#include <iostream>
#include <cstdio>
#include <algorithm>
#include <unordered_map>

using namespace std;

const int mod = 1e9 + 7;

int n;
unordered_map<int, int> mp;

int main()
{
    cin >> n;
    while(n --)
    {
        int x;
        cin >> x;
        
        for(int i = 2; i <= x / i; i ++)
        while(x % i == 0)
        x /= i, mp[i] ++;
        
        if(x > 1)
        mp[x] ++;
    }
    
    long long res = 1;
    for(auto t : mp)
    res *= t.second + 1, res %= mod;
    
    cout << res;
    
    return 0;
    
}
```

### 三、约束之和

x的所有约数之和


----------

### 算法
将x分解为p1^a1 · p2^a2 · p3^a3...形式
pi表示最小的质因子
x的任意一个约束都可分解为p1^x1 · p2^x2 · p3^x3... xi取值[0, a1]
即x的约数之和为
(p1^0+p1^1+...+p1^a1)(p2^0+p2^1+..+p2^a2)...(pk^0+pk^1+...+pk^ak)
### $O(nlog(a))$
#### C++ 代码
```cpp
#include <iostream>
#include <cstdio>
#include <cstring>
#include <unordered_map>
#include <algorithm>

using namespace std;

const int mod = 1e9 + 7;

int n;
unordered_map<int, int> mp;

int main()
{
    cin >> n;
    
    while(n --)
    {
        int x;
        cin >> x;
        
        for(int i = 2; i <= x / i; i ++)
        {
            while(x % i == 0)
            mp[i] ++, x /= i;
        }
        if(x > 1)
        mp[x] ++;
    }
    
    long long res = 1;
    for(auto t : mp)
    {
        int p = t.first, cnt = t.second;
        long long ans = 1;
        while(cnt --)
        ans = ans * p + 1, ans %= mod;
        
        res *= ans, res %= mod;
    }
    cout << res;
    
    return 0;
}
```

### 四、最大公约数

a b最大公约数
----------

### 算法
已知d | a, d | b, 则必然 d | (ax + by)
欧几里得公式 gcd(a, b) == gcd(b, a % b)
证明：
a % b == a - (a / b) * b == a - c·b
左侧任意的公约数d, 都有 d | a, d | b
则必然 d | a - c·b 即 d | a % b
即所有左侧公约数，右边都有
反之亦然
即二者等价

### $O(log(n))$
#### C++ 代码
```cpp
#include <iostream>
#include <cstdio>

using namespace std;

int n;

int gcd(int a, int b)
{
    return b ? gcd(b, a % b) : a;
}

int main()
{
    cin >> n;
    
    while(n --)
    {
        int a, b;
        cin >> a >> b;
        cout << gcd(a, b) << endl;
    }
    return 0;
}
```
