### 一、质数判定

快速判定数字是否为质数
质数和合数的定义是
从2开始，只能被1和它自身整除的数为质数，否则为合数


#### 样例

```
2最小的质数，4最小的合数
```


----------

### 算法 试除法
数x有若干个质因子相乘得到，若d可以整除x，则x/d也可以整除x，这样的一对因子成对出现，因此我们只需要试除2~sqrt(x)之间的数字即可
### $O(sqrt(n))$
#### C++ 代码
```
#include <iostream>
#include <cstdio>

using namespace std;

bool isprime(int x)
{
    if(x < 2) return false;     //从最小的数开始
    for(int i = 2; i <= x / i; i ++)    //这样设计判断条件，不会爆int
    {
        if(x % i == 0) return false;
    }
    return true;
}

int main()
{
    int n;
    cin >> n;
    while(n --)
    {
        int x;
        cin >> x;
        if(isprime(x))
        cout << "Yes";
        else
        cout << "No";
        cout << endl;
    }
    return 0;
}
```

### 二、分解质因数

将x分解为p1^x1 · p2^x2 · p3^x3...形式
pi表示最小的质因子

#### 样例

```
18 = 2^1 · 3^2
```


----------

### 算法 试除法

### $O(sqrt(n))$
特别的 当n为2^k时，时间复杂度为 $O(log(n))$
#### C++ 代码
```
#include <iostream>

using namespace std;

int n;

void decompose(int x)
{
    for(int i = 2; i <= x / i; i ++)        //大于sqrt(x)的因子最多有一个，只分解前sqrt(x)个数，剩下的数单独处理
    {
        if(x % i == 0)      //每个i都被前2~i-1的所有数除干净，因此if成立时，i一定是质数
        {
            int s = 0;
            while(x % i == 0)
            s ++, x /= i;
            cout << i << " " << s << endl;
        }
    }
    if(x > 1)   //x就是大于sqrt(原x数)的唯一因子
    cout << x << " " << 1 << endl;
}

int main()
{
    cin >> n;
    while(n --)
    {
        int x;
        cin >> x;
        decompose(x);
        cout << endl;
    }
    return 0;
}
```

### 三、筛质数

统计1~n中质数

----------

### 算法1 线性筛
核心：x只会被最小质因子筛掉

### $O(n)$
#### C++ 代码
```
#include <iostream>

using namespace std;

const int N = 1e6 + 10;

int n;
int primes[N], idx;
bool st[N];

void choose(int x)
{
    for(int i = 2; i <= x; i ++)
    {
        if(!st[i]) primes[idx ++] = i;
        for(int j = 0; primes[j] <= x / i; j ++)
        {
            st[primes[j] * i] = true;
            if(i % primes[j] == 0) break;
        }
    }
}

int main()
{
    cin >> n;
    
    choose(n);
    
    cout << idx;
    
    return 0;
}
```

### 算法2 埃氏筛

### $O(nloglogn)$
#### C++ 代码
```
#include <iostream>
#include <cstdio>

using namespace std;

const int N = 1e6 + 10;

int n;
int primes[N], idx;
bool st[N];

void choose(int x)
{
    for(int i = 2; i <= x; i ++)
    {
        if(!st[i]) 
        {
            primes[idx ++] = i;
            for(int j = i + i; j <= x; j += i)
            st[j] = true;
        }
    }
}

int main()
{
    cin >> n;
    
    choose(n);
    
    cout << idx;
    return 0;
}
```
