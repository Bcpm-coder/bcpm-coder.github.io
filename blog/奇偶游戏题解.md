# 奇偶游戏题解：带权并查集与离散化

[AcWing 239 · 奇偶游戏](https://www.acwing.com/problem/content/description/241/)

### 题目描述

给你多个区间，每个区间中有奇或偶数个1
请你给出从哪个地方开始出现矛盾
#### 样例
10
5
1 2 even
3 4 odd
5 6 even
1 6 even
7 10 odd
```text
1~2     偶数个1
3~4     奇数个1
5~6     偶数个1
1~6     偶数个1
显然前三条说明了1~6中有奇数个1，第四条与之矛盾
```
----------

### 算法
##### (并查集)  
利用前缀和的思想，给你区间l ~ r这一段数字和的奇偶性
等价于s[r] - s[l - 1]的奇偶性 （s表示前x个数中有多少个1）
等价于s[r]和s[l - 1]这两个数是否为同一类数，奇性则两数为异类，偶性两数为同类
利用并查集方便实现，同时维护一个d[x]数组，保存!:x到p[x]的关系，为0表示与p[x]同类，为1表示与p[x]异类

#### 具体细节
先求出两点的祖宗节点
若不同，则进行合并：
两祖宗节点的合并，要保证合并后x到y的距离根据给定的奇偶性做出奇性或偶性的赋值
若相同，则要判断两者关系和给定奇偶性是否冲突
不冲突则进行合并
#### C++ 代码
```cpp
#include <iostream>
#include <cstring>
#include <cstdio>
#include <unordered_map>

using namespace std;

const int N = 10010;

int n, m, res;
int p[N], d[N];
int idx;
unordered_map<int, int> mp;

int get(int x)                  // 哈希表映射大数
{
    if(mp.count(x) == 0) 
    mp[x] = idx ++;
    return mp[x];
}

int findp(int x)
{
    if(p[x] != x)
    {
        int r = findp(p[x]);
        d[x] += d[p[x]];        // 维护这个距离，长度无所谓，主要是奇偶性
        p[x] = r;
    }
    return p[x];
}

int main()
{
    cin >> n >> m;
    
    for(int i = 1; i < N; i ++) //  记得初始化
    p[i] = i;
    
    for(int i = 0; i < m; i ++)
    {
        char op[5];
        int a, b;
        cin >> a >> b >> op;
        int x = get(a - 1), y = get(b);
        
        int px = findp(x), py = findp(y);
        
        int t = 1;
        if(*op == 'e')
        t = 0;
        else if(*op == 'o')
        t = 1;
        // cout << px << " " << py << " " << d[x] << " " << d[y] << " " << t << endl;
        res = m;
        if(px == py && (d[x] + d[y]) % 2 != t)      // 要判断两者关系和给定奇偶性是否冲突
        {
            res = i;
            break;
        }
        else        // 祖宗不同 或 不冲突则进行合并
        {
            d[px] = (d[x] + d[y] + t) % 2;
            p[px] = py;
        }
        
    }
    cout << res;
    
    return 0;
}
```
