# 构建个人主页的一些记录

这次个人主页没有采用传统的单页作品集布局，而是把内容组织成一个可以交互的桌面环境。

## 内容组织

- **关于我**：姓名、头像与教育经历
- **项目**：工程项目、技术栈与项目亮点
- **技能**：保留清晰的技术分类
- **音乐**：提供独立的沉浸式播放器
- **博客**：用 Markdown 持续沉淀内容

## 为什么使用 Markdown

Markdown 足够轻量，文章本身仍然是普通文本文件，便于使用 Git 管理，也不会被某个特定编辑器绑定。

    写作 → 保存 Markdown → 更新文章清单 → 发布

## 继续完善

未来可以加入文章目录、代码高亮、阅读进度和更多分类，但最重要的仍然是持续写下真正有价值的内容。
