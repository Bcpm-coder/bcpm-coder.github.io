# 欢迎来到我的博客

这里用于记录学习过程、项目实践和偶尔冒出的想法。文章使用 **Markdown** 编写，页面会自动将文档转换成适合阅读的内容。

## 如何添加新文章

1. 将新的 **.md** 文件放入 **src/assets/blog/posts/**。
2. 打开 **src/assets/blog/posts.json**。
3. 按照现有格式添加标题、日期、分类、标签和文件路径。分类可填写 **Coding**、**学习**、**生活** 或 **碎碎念**。
4. 重新构建网站，文章就会出现在左侧列表中。

    {
      "id": "my-new-post",
      "title": "我的新文章",
      "excerpt": "一句简短的文章摘要。",
      "date": "2026-07-20",
      "readTime": "5 分钟阅读",
      "category": "学习",
      "tags": ["学习笔记"],
      "file": "/assets/blog/posts/my-new-post.md"
    }

> Markdown 内容会经过安全清理后再显示，可以放心使用常见的标题、列表、引用、链接、图片和代码块。
